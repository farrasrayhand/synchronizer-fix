# Script Collection

Kumpulan skrip otomatis oleh Farras Rayhand.

## Struktur Repositori

Repositori ini diatur sebagai berikut:

- `/scripts`: Berisi file skrip mentah (PowerShell, Bash, dll).
- `/web`: Berisi halaman tutorial/antarmuka web untuk setiap skrip.
- `index.html`: Portal utama untuk mengakses semua tutorial skrip yang tersedia.

## Daftar Skrip

### 1. eRapor SMK Synchronizer Fix Install
Skrip untuk fix error saat instalasi eRapor SMK Synchronizer.
- **Skrip**: `scripts/synchronizer-fix.ps1`
- **Tutorial**: [Lihat Tutorial](https://script.minicenter.my.id/web/synchronizer-fix/)

### 2. PHP Imagick Auto Installer
Skrip otomatis untuk install ekstensi PHP Imagick di Laragon dan Laravel Herd.
- **Skrip**: `scripts/imagick-autoinstaller.ps1`
- **Tutorial**: [Lihat Tutorial](https://script.minicenter.my.id/web/imagick-autoinstaller/)

## Cara Kontribusi

1. Fork repositori ini.
2. Simpan skrip di folder `/scripts/[nama-skrip]`.
3. Buat folder baru di `/web/[nama-skrip]` untuk tutorial/halaman webnya.
4. Perbarui `index.html` utama dan `README.md`.
5. Kirim Pull Request.
