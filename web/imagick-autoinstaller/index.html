PHP Imagick Auto Installer

Skrip otomatis untuk install ekstensi PHP Imagick di Laragon dan Laravel Herd (Windows).

1

Buka Terminal (Admin)

Gunakan PowerShell (Direkomendasikan) atau Command Prompt (CMD).

2

Salin Perintah

Pilihan 1: Standard Install

Recommended
irm http://script.minicenter.my.id/scripts/imagick-autoinstaller.ps1 | iex

Gunakan ini untuk instalasi baru atau menambah Imagick ke versi PHP yang belum terpasang.

Pilihan 2

Force / Repair Mode

Gunakan ini jika PHP error saat startup (Startup Error), atau jika Anda ingin memaksa install ulang (overwrite) file yang sudah ada.

PowerShell Only
$s=irm http://script.minicenter.my.id/scripts/imagick-autoinstaller.ps1; &([scriptblock]::Create($s)) -Force

Pilihan 1: Standard Install

CMD Wrapper
powershell -ExecutionPolicy Bypass -Command "irm http://script.minicenter.my.id/scripts/imagick-autoinstaller.ps1 | iex"

Gunakan ini untuk instalasi baru via Command Prompt.

Pilihan 2

Force / Repair Mode

Gunakan ini untuk memperbaiki error startup via Command Prompt.

CMD Wrapper
powershell -ExecutionPolicy Bypass -Command "$s=irm http://script.minicenter.my.id/scripts/imagick-autoinstaller.ps1; &([scriptblock]::Create($s)) -Force"
3

Jalankan

Paste ke PowerShell lalu tekan ENTER.

Skrip ini akan mendeteksi semua versi PHP yang terpasang di Laragon dan Laravel Herd secara otomatis.

Pastikan koneksi internet aktif saat menjalankan skrip.
