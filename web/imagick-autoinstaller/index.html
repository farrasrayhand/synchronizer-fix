PHP Imagick Auto Installer

Skrip otomatis untuk install ekstensi PHP Imagick di Laragon dan Laravel Herd (Windows).

1

Buka PowerShell (Admin)

Klik kanan Start > Terminal (Admin) atau PowerShell (Admin).

2

Salin Perintah

Gunakan perintah standar untuk instalasi baru:

Standard Install
powershell -ExecutionPolicy Bypass -Command "irm http://script.minicenter.my.id/scripts/imagick-autoinstaller.ps1 | iex"
Force Mode

Perbaikan Otomatis

Gunakan ini jika Imagick sudah terpasang tapi error (Startup Error), atau jika Anda ingin memaksa install ulang di semua versi.

Force / Repair
powershell -ExecutionPolicy Bypass -Command "$s=irm http://script.minicenter.my.id/scripts/imagick-autoinstaller.ps1; &([scriptblock]::Create($s)) -Force"
3

Jalankan

Paste ke PowerShell lalu tekan ENTER.

Skrip ini akan mendeteksi semua versi PHP yang terpasang di Laragon dan Laravel Herd secara otomatis.
